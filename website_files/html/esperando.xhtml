Encuéntralo

Instrucciones

¿Te gusta buscar objetos ocultos? "Encuentralo" consiste en buscar los objetos que tienes en tus cartas. Debes ser más rápido que tus amigos y afinar tu visión
para encontrar todos los objetos rápido. Divertete con los siguientes modos de juego:

Tradicional

A cada jugador se le asignan 3 dibujos de manera aleatoria en cada ronda y los busca en el área
de juego, la ronda termina cuando un jugador encuentra los 3 dibujos.

Compartidas

Se le asignan 3 figuras elegidas aleatoriamente a todos los jugadores y el primero en encontrar las 3
es el ganador de la ronda.

Este es un juego creado para el curso de Diseño de aplicaciones Web con el profesor Jeisson Hidalgo
durante el segundo semestre del año 2020. Este se usó para aprender sobre diseño web.
